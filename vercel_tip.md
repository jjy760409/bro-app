# Vercel 환경 변수 입력 시 주의사항

**결론부터 말씀드리면: 따옴표(")는 입력하지 마세요!**

Vercel의 환경 변수 입력 칸(Value)에는 **순수한 값**만 넣어야 합니다.

### ❌ 잘못된 예 (따옴표 포함)
*   Key: `VITE_FIREBASE_API_KEY`
*   Value: `"AIzaSyD..."`  <-- **(X) 따옴표 넣으면 안 됨**

### ✅ 올바른 예 (값만 입력)
*   Key: `VITE_FIREBASE_API_KEY`
*   Value: `AIzaSyD...`  <-- **(O) 키 값만 깔끔하게**

---

### 🚀 추가 업데이트 알림
아까 말씀드린 **'글로벌 차별화 기능(다국어, 식이요법)'** 코드들이 아직 **고객님의 컴퓨터에만 있고 GitHub에는 올라가지 않은 상태**였습니다. (그래서 `Everything up-to-date`라고 떴던 겁니다.)

제가 지금 바로 이 기능들을 **저장(Commit)하고 업로드(Push)** 해드리겠습니다.
잠시 후 Vercel이 이걸 감지하면 **진짜로 업데이트**가 시작됩니다.
